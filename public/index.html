Url Shortener

A back-end microservice project from FreeCodeCamp

documentation:

This service will take a web address and return a shortened unique id for that particular website

  • Add a website address by entering a valid web address using http:// or https:// and click on Add Link
  • When you click the Add Link button, you will recieve a message stating that your custom web address was successfully saved to the database.
  • Press the Get Id button to recieve a unique shortened id
  • Take the id given and add it to the original homepage link like so: https://www.localhost.com/ENTER-CUSTOM-ID-HERE
